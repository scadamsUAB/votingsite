Welcome to the Voting DAPP.



This application is connected to the Songbird Network Smart Contract to handle voting based on the ownership of Pixel Plant NFTs

>How It Works

The Smart contract has been loaded with the addresses of all Pixel Plant NFT holders. Only these individuals are elligible to vote.

In addition, If you hold more than one NFT, you have more voting power (i.e. if you have 2 your vote counts as 2 votes, if you have 3 it counts as 3 votes etc..)

Only one selection can be made per user and all of your votes will go to that selection.

When voting closes, the winning option will be selected.

This vote is to determine what charity the initial Pixel Plant Mint for April will receive the donation.

Below is a list of the charity candidates and a link to their profile on Charity Navigator
  • Pure Earth
  • Clean Ocean Action
  • Earth Day Network
  • EarthJustice
  • Ecotrust


Select an option from the drop down and click submit. Sign the transaction and you are Done.
